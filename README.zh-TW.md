# Weather
[English](./README.md) | 正體中文

一個命令列介面的 Python 應用程式，使用 [OpenWeatherMap](https://openweathermap.org/) API 來取得任何城市的即時天氣資料。

## Features
- 取得目前溫度（攝氏）
- 天氣狀況（晴天、雨天、多雲等）
- 查看最高和最低溫度
- 適用於全球任何城市

## Installation
無需安裝其他東西，從 [Releases page](https://github.com/monosdev/weather/releases) 下載最新版本並直接執行。

## Running (Linux)
1. 移動到下載資料夾。

2. 讓程式具有可執行性：
```bash
chmod +x app
```

3. 執行程式：
```bash
./app
```
